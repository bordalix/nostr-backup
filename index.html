Nostr backup

Fetch and backup your events.

Know more

This app will fetch your events from a big list of relays and generate a javascript file for your backup.

If you want to use your own list of relays:

  • Clone repo
  • Edit js/relays.js
  • Open index.html with a browser

Repo at https://github.com/bordalix/nostr-backup
